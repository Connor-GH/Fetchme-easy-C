
## Current Features

- color (not automatic yet, but it is customizable)
- username@hostname
- distro name
- kernel
- uptime
- package count
- shell
- resolution
- refresh rate
- wm name
- terminal name
- cpu name
- cpu thread count
- cpu temp
- cpu frequency
- gpu name
- memory usage
- memory percent
- disk usage
- disk percent

- ascii logo (same distro.h dependency)
- package count working on more distros
- cpu temperature working on more cpus
- disk usage working on more than just the root partition
